# Unified Messenger & Productivity App

A cross-platform productivity app integrating messaging, collaborative notes, todos, calendar, and email.

## Features

- Messaging (Matrix)
- Collaborative Notes
- Todo List
- Calendar
- Email

## Tech Stack

- Svelte
- TypeScript
- Vite
- Tauri
- Docker

## Getting Started

1.  `npm install`
2.  `npm run dev`
